데이터의 분류

	정성적 데이터 : 숫자로 표현되는 데이터 수치 - 비정형 데이터, 요약, 주관적 결론
	정량적 데이터 : 자연언어에 의한 서술로 표현되는 범주형 데이터 - 정형데이터, 통계 분석, 객관적 결론

	- 형태별로 분류
	정형 데이터 : 고정된 필드에 저장된 데이터. 관계형 데이터베이스를 예로 들 수 있다.
		설계한 기술자에 의해 수집되는 정보의 형태가 정해지게 된다. 
	반정형 데이터 : 고정된 필드에 저장된 데이터는 아니지만 XML, HTML 텍스트 등 메타데		이터 및 스키마를 포함하는 데이터이다.
	비정형 데이터 : 고정된 필드에 저장되어 있지 않은 데이터를 의미하며 페이스북과 트위터, 유튜브 영상, 이미지 파일등이 그 예이다. 

	암묵지 - 학습과 경험을 통하여 습득함으로써 개인에게 체화되어 있지만 언어나 문자로 표현하기 어려운, 겉으로는 드러나지 않는 지식을 말한다. 경험을 통해 얻기 때문에 경험지 하고도 한다.
	형식지 - 언어나 문자를 통하여 겉으로 표현된 지식으로서 문서화 또는 데이터화 된 지식.

	MYSQL (MYSQL) - 중간 결과 저장
	PLAY (PLAY FRAMEWORK) - DASHBOARD 제작
		
	하둡 (HADOOP) - 방대한 향의 정형/비정형 데이터를 분산 처리하여 빠른 시간내에 결과를 제공하는 오픈소스 기반의 데이터 관리 기술. 
	구글의 빅쿼리(BIG QUERY) - 빅데이터를 클라우드 상에서 신속하게 분석해주는 서비스. 이용자가 구글 클라우드 스토리지에 분석하고자 하는 데이터를 업로드 하면 웹 브라우저를 통해 데이터가 분석된다. 별도의 인프라를 구축하지 않아도 데이터를 분석할 수 있다.
	아마존의 다이나모 - 아마존은 차세대 가상 분산 저장공간 시스템을 개발. 키-밸류의 형태로 대용량 데이터를 저장할 수 있으며 고속의 데이터 접근 가능. 
	APACHE SPARK - 하둡과 마찬가지의 클러스터 컴퓨팅 프레임웍. 메모리를 활용한 아주 빠른 데이터 처리가 특징. 함수형 프로그래밍이 가능한 언어인 SCALA를 사용하여 코드가 매우 간단하며, INTERACTIVE SHELL을 사용할 수 있다.
	SMART FACTORY - 공장 내 설비와 기계에 센서(IOT)가 설치되어 데이터가 실시간으로 수집, 분석되어 공장내 모든 상활들이 일목요연하게 보여지고(OBSERVABILITY)고, 이를 분석해 목적된 바에 따라 스스로 제어(CONTROLLABILITY)되는 공장을 말한다.
	-> 생산자동화와 스마트팩토리 : 현재 생산현장들은 단위 공정별로만 자동화, 최적화가 이루어짐. 공정과 공정이 유기적으로 연계되어 있이 않음. 스마트 팩토리가 구현되면 의사결정하는 데이터 기반의 공장 운영체계를 갖춤으로써 생산현장에서 발생하는 현상, 문데제들의 상관관계를 얻어낼 수 있다.
	MACHINE LEARNING - 기계가 직접 데이터를 학습함으로써 그 속에 숨겨진 일련의 규칙성을 찾는다.
	DEEP LEARNING - 딥러닝이란 기계학습의 일부로 기게학습과 대부분의 것들이 매우 비슷하지만 일반적인 기계학습과 두가지의 큰 특징을 뽑으면 다음과 같다.
		데이터의 특징을 사람이 추출하지 않는다. 데이터 전체를 학습시킨다.
		주로 인공신경망 구조를 사용하여 학습한다.

	텍스트 마이닝 - 비/반정형 텍스트 데이터에서 자연어 처리 기술에 기반하여 유용한 정보를 추출, 가공하는 것을 목적으로 하는 기술
	-> 응용 분야로 문서분류, 문서군집, 정보추출, 문서요약 들이 있다.

	오피니언 마이닝 - 텍스트 마이닝의 관련 분야로는 오피니언 마이닝, 혹은 평판 분석(SENTIMENT ANALYSIS)이라고 불리는 기술이 있다. 오피니언 마이닝은 소셜미디어 등의 정형/비정형 텍스트의 긍적, 부정, 중립의 선호도를 판별하는 기술이다.

	소셜 네트워크 분석 - 수학의 그래프 이론에 뿌리를 두고 있다. 소셜 네트워크 연결구조 및 연결강도 등을 바탕으로 사용자의 명성 및 영향력을 측정한다.

	클러스터 분석(군집 분석) - 비슷한 특성을 가진 개체를 합쳐가면서 최종적으로 유사한 특성의 군집을 발굴하는데 사용된다. 예를 들어 트위터 상에서 주로 사진/카메라 등 관심사나 취미에 따른 사용자군 군집분석을 통해 분류할 수 있다.

딥러닝 분석 기법의 종류와 오픈소스의 종류

	LSTM(LONG SHORT-TERM MEMORY) - RNN의 일종. RNN의 히든 스테이트에 CELL-STATE를 추가한 구조.
	AUTOENCODER - 비감독 방식으로 효율적인 데이터 코딩을 학습하기 위해 사용되는 인공신경망의 일종이다.
	K-NN(K NEAREST NEIGHBORHOOD) - 분류나 회구에 사용되는 방식. 인스턴스 기반 학습. - 머신러닝 기법
	RNN(RECURRENT NEURAL NETWORK) - 히든 노드가 방향을 가진 엣지로 연결돼 순환 구조를 이루는 인공신경망의 한 종류. 음성, 문자 등 순차적으로 등장하는 데이터 처리에 적합한 모델. CNN(CONVOLUTIONAL NEURAL NETWORKS)와 더불어 각광 받고 있는 알고리즘. 시퀀스 길이에 관계없이 인풋과 아웃풋을 받아들일 수 있는 네트워크의 구조로 필요에 따라 다양하고 유연하게 구조를 만들 수 있다.

셀프 서비스 분석(SELF SERVICE ANALYTICS)

	IT 지원을 통해 기간업무 전문가가 쿼리를 수행하고 자체적으로 보고서를 생성할 수 있도록 하는 BI(비즈니스 인텔리전스)의 한 형태입니다. 셀프 서비스 분석은 기본 분석 기능을 갖춘 사용하기 쉬운 BI 도구와 이해하기 쉽고 직접적인 데이터 액세스를 위해 단순화 또는 축소 된 기본 데이터 모델을 특징으로합니다.
